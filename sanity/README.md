# BRASA Sanity Studio

This Studio powers editable content for the Next.js platform.

## Run locally

```bash
npm install
npm run dev
```

## Required documents

Create these documents (singletons by slug):

- `siteShell` with `slug = default`
- `homePage` with `slug = home`
- `pricingPage` with `slug = pricing`

Create `libraryItem` documents for the library page.
